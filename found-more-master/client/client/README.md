# found-more-frontend
Front-End for the Found MORE system. 

## Set up
Install all the packages
```
npm install
```

Run the development server (webpack)
```
npm run dev
```

Do ya thang!

## Technologies used
1. React
2. Redux
3. Redux-Thunk
3. GraphQL (in combination with Apollo Client)
4. PostCSS
5. Autoprefixer
